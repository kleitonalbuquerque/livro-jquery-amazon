Função addClass

A função addClass() permite adicionar uma classe CSS a um determinado elemento. É importante destacar que essa classe deve existir no arquivo CSS ou na seção style, caso contrário a função não terá nenhum efeito.

Função hasClass

Essa função hasClass() permite verificar se determindo elemento possui uma classe CSS.

Função removeClass

A função removeClass() faz exatamente o oposto da função addClass() ou seja, remove uma classe de determinado elemento.

Função toggleClass

A operação de adicionar e remove classes CSS é algo bem comum no jQuery - minha função preferida.

Função hide

Esta função esconde/oculta um elemento, com isso, aplica a propriedade "display: none"

Função show

Esta função show() faz o oposto da função hide(). Ao clicar no botão abaixo ele mostrará o footer caso o mesmo esteja ocultado.

Função attr

A função attr() permite alterar um atributo de um elemento.
